# Oculus Picture View
=====================

## ヘッドマウントディスプレイであるOculus Riftを使用して画像をスライドショー表示するプログラムです。

- OculusPictureView_DirectToRift.exe を実行してください。
  + Oculus Riftがないと起動しない場合があります。
  + Ecsキーで終了します。

- imgフォルダに入れた.jpgまたは.pngファイルを10秒毎に切り替えます。
  + 透過PNGには対応できませんでした…申し訳ありません。

- キーボードの上下左右で移動して、下から見上げたりすることもできます。

- 動きすぎると無限落下するのは仕様です。復帰方法はありませんので立ち上げ直してください。
